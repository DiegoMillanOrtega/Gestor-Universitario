# Gestor-Universitario

# Declaración de Alcance

## 1.1. Definición de Problema/Oportunidad
El software busca solucionar la falta de organización académica que enfrentan los estudiantes durante el semestre. Problemas como olvidar fechas de exámenes y entregas, mantener materiales desordenados y no tener claridad sobre las tareas semanales afectan el control académico. El sistema centralizará las actividades, ordenará los recursos y enviará recordatorios para mejorar la gestión del semestre.

## 1.2. Definición del Producto
Al finalizar el periodo de desarrollo, el proyecto debe ser capaz de registrar las asignaturas del semestre, permitir el registro de tareas asociadas a cada materia y mostrar una lista organizada y accesible de todas las tareas pendientes por asignatura.
